One Dependency to Rule Them All

Minimal Local Dependencies (cross platform, as much as possible)

One package to install, all the things provisioned on demand.


  name: cornell-camp-sprint
  recipe: pantheon
  config:
    framework: drupal8
    env: dev
    site: cornell-camp-sprint
    id: ec66283b-0afa-4b2c-a884-34d4e999ee3e
  services:
    appserver:
      overrides:
        services:
          environment:
            BEHAT_PARAMS: >-
            {"extensions" : {
                "Behat\\MinkExtension" : {
                    "base_url" : "http://nginx/"}}}
  tooling:
    test:
      service: appserver
      cmd:
        - "composer"
        - "test"
    phpunit:
      service: appserver
      cmd:
        - "composer"
        - "test:unit"

Single declarative file to configure infrastructure that ships with the project
